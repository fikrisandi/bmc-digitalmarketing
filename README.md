## Cara Deploy ke Vercel

### Paling Mudah: Drag & Drop
1. Buka vercel.com/new
2. Drag & drop folder ini
3. Framework: Pilih "Other"
4. Klik Deploy
5. Selesai!

### Via CLI
```bash
npm install -g vercel
vercel login
vercel
vercel --prod
```
